# Thoughtful AI Support Agent

Simple CLI that answers common questions about Thoughtful AI using hardcoded responses.


## Development

```bash
uv venv .venv
uv pip install -e .
```

## Usage

```bash
./.venv/bin/thoughtful-ai-agent "What does the eligibility verification agent (EVA) do?"
./.venv/bin/thoughtful-ai-agent "What does the claims processing agent (CAM) do?"
./.venv/bin/thoughtful-ai-agent "How does the payment posting agent (PHIL) work?"
./.venv/bin/thoughtful-ai-agent "Tell me about Thoughtful AI's Agents."
./.venv/bin/thoughtful-ai-agent "What are the benefits of using Thoughtful AI's agents?"
```
